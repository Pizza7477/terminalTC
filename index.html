Enumeration Flow

1

Discover hosts

Goal: find live machines on the network.

Example: sudo netdiscover -r 10.38.1.0

↓
2

Verify target is alive

Goal: make sure the host responds.

Example: ping -c 3 10.38.1.117

↓
3

Scan ports & services

Goal: see which services are open.

Example: nmap -sV -O 10.38.1.117

↓
4

Attack login / passwords

Goal: try leaked passwords on the login page.

Example: hydra -l hari -P hackme.txt 10.38.1.117 ...

↓
5

Privilege escalation

Goal: find SUID binaries & paths to root.

Example: find / -perm -4000 -type f -exec ls -l {} \; 2>/dev/null
